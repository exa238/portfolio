Disney Trivia Experience

About the Project

Disney Trivia Experience is an interactive game build in Unity to explore how playful questions, themed UI and rapid feedback loops can create engagement. the project focused on designing a smooth question and answer flow, building a responsive UI system, and balancing charming and nostalgic aesthetics with functional clarity.

Project Goals

  • Create an intutive trivia loop that keeps players focused on the challenge
  • Use Disney-inspured visuals to enhance immersion without overwhelming the user
  • Test how UI feedback (correct/incorrect states, score progression) shapes user engagement

Core Features

  • Question Display: Presents each prompt with clear hierarchy and minimal distraction.
  • Answer Input System: Buttons and interactions designed for quick, confident selection.
  • Feedback States: Immediate visual responses for correct/incorrect answers to reinforce learning.
  • Score and Results Screen: Summarizes performance and encourages replayability.

Outcome

  • Screen-recorded demo showcased in class and used for peer testing.
  • Strengthened my ability to balance theme, clarity, and game flow, ensuring the UI supports the experience rather than competing with it.
